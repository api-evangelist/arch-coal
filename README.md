## Features

| Name | Description |
|------|-------------|
| Metallurgical Coal | High-quality metallurgical coal for steelmaking from mines in West Virginia and Virginia. |
| Thermal Coal | Thermal coal for electricity generation from mines in Wyoming's Powder River Basin. |
| Safety Performance Data | Publicly reported mine safety and environmental performance metrics. |
| Production Reporting | Quarterly coal production and sales volume reporting for investors and analysts. |
| SEC Filings | Annual reports, 10-K, 10-Q, and 8-K filings available through SEC EDGAR. |
